# Redes de Computadores

## O que é uma rede?

Uma rede de computadores permite que dispositivos se comuniquem e compartilhem informações.

## Conceitos básicos

- Endereço IP
- Máscara de Rede
- Gateway
- DNS
- DHCP

## Protocolos

- TCP/IP
- HTTP
- HTTPS
- SSH
- RDP

## Equipamentos

- Switch
- Roteador
- Firewall
- Access Point

## Comandos úteis no Windows

```cmd
ipconfig
```

Mostra as configurações da rede.

```cmd
ping google.com
```

Verifica se existe comunicação com outro computador ou site.

```cmd
tracert google.com
```

Mostra o caminho percorrido até o destino.

```cmd
nslookup google.com
```

Consulta o servidor DNS.
