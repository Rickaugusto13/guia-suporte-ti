# 👥 Active Directory para Suporte Técnico

## Objetivo

Este documento reúne conceitos, tarefas do dia a dia e boas práticas
relacionadas ao Active Directory, com foco em atividades comuns de
Analistas de Suporte.

## O que é o Active Directory?

O Active Directory (AD) é o serviço de diretório da Microsoft utilizado
para centralizar o gerenciamento de usuários, computadores, grupos e
permissões em um ambiente corporativo.

## Ferramentas utilizadas

-   Active Directory Users and Computers (ADUC)
-   Group Policy (GPO)
-   Remote Desktop (RDP)
-   Windows Server

## Atividades comuns

-   Criar usuários
-   Redefinir senhas
-   Desbloquear contas
-   Habilitar e desabilitar usuários
-   Adicionar e remover usuários de grupos
-   Organizar contas em Unidades Organizacionais (OUs)

## Checklist de atendimento

-   Confirmar a identidade do usuário
-   Verificar se a conta está bloqueada
-   Validar grupos de acesso
-   Testar o login após a alteração
-   Registrar o atendimento

## Boas práticas

-   Aplicar o princípio do menor privilégio.
-   Nunca compartilhar senhas.
-   Registrar todas as alterações realizadas.
-   Validar solicitações antes de conceder acessos.
