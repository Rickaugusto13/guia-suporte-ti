## 📖 Sobre o Projeto
Este repositório é o meu **Portfólio Profissional** e Base de Conhecimento técnica. Ele reúne anotações, guias práticos e registros de experiências adquiridas ao longo de mais de 16 anos atuando em infraestrutura, suporte e sustentação de sistemas críticos, com forte foco no mercado financeiro [1, 5, 7].

O objetivo é documentar processos e ferramentas que dominam o dia a dia de um **Analista de Sistemas Sênior**, servindo como guia de estudo e demonstração de competências técnicas.
